# WordPress.org Plugin Assets

Place plugin directory assets in this folder:

- `banner-772x250.png` – Plugin banner (required)
- `banner-1544x500.png` – High-DPI plugin banner (optional)
- `icon-128x128.png` – Plugin icon (recommended)
- `icon-256x256.png` – High-DPI plugin icon (recommended)
- `screenshot-1.png` – Screenshot referenced in readme.txt (optional)

These images are displayed on the WordPress.org plugin page and are
deployed to the SVN `/assets/` directory by the deploy workflow.
